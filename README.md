# 📊 Startup Financial KPI Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing the financial performance of an early-stage startup using key business metrics and interactive dashboards. The analysis helps evaluate startup growth, customer profitability, sustainability, and financial efficiency.

The project was developed as part of the **Elevate Labs Internship Final Project** using **Excel, Tableau, and Python (Pandas)**.

---

## 🎯 Objective
To analyze startup financial performance using KPIs such as:

- Revenue
- Expenses
- Burn Rate
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV)
- LTV:CAC Ratio
- Run Rate
- Churn Rate
- Customer Cohort Analysis

The goal is to understand startup profitability, customer retention, and operational sustainability through data-driven insights.

## 📈 KPIs Analyzed

### 1. Revenue Trend Analysis
Tracks startup revenue growth over time to understand business performance trends.

### 2. Burn Rate Trend Analysis
Measures the speed at which the startup spends capital over time.

### 3. Customer Acquisition Cost (CAC) Analysis
Analyzes the cost of acquiring customers across industries.

### 4. Lifetime Value (LTV) Analysis
Measures long-term customer profitability.

### 5. LTV:CAC Ratio Analysis
Evaluates customer acquisition efficiency.

**Formula:**

```text
LTV:CAC Ratio = LTV / CAC
```

Interpretation:

- **> 3** → Healthy business model
- **1 – 3** → Moderate performance
- **< 1** → Unsustainable customer acquisition

### 6. Run Rate Analysis
Projects annualized business performance based on current financial trends.

### 7. Churn Rate Analysis
Measures customer loss percentage across industries.

### 8. Customer Cohort Analysis
Groups customers by time period to understand retention patterns.
